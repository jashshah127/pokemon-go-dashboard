# Pokemon GO Player Dashboard

Interactive analytics dashboard built with Python and Plotly Dash, transforming a raw Pokemon GO personal data export into insights.

## Features
- Overview, Pokemon Collection, Encounters, Live Map, Exploration, Raids, Fitness, Spending, Social

## Tech Stack
Python, Pandas, Plotly Dash, Folium

## Setup
pip install dash plotly pandas folium
python app.py

Open http://localhost:8050
